# Music Player

A new Flutter project. In which it access the audio files which are on the device and shows them as a list. 

## Features

- Play/Pause the song
- Play next song on on the list
- Autoplay next song after completion of current playing.

## Dependecies Used

- device_info_plus
- permission_handler
- provider
- on_audio_query
- audioplayers
